Czym jest tag <article>?

Element **<article>** reprezentuje samodzielną, niezależną treść. Dodany CSS sprawia, że strona jest czytelniejsza i bardziej atrakcyjna wizualnie. Bez stylów, elementy są po prostu układane jeden pod drugim, używając domyślnej czcionki przeglądarki.

To jest przykładowy element z użyciem klasy **.alert-warning** dla wizualnego ostrzeżenia.
